# AI Instructions

This folder contains instructions, prompts, and configurations for AI assistants working on the AI Context Service project.

## Purpose
Provides AI systems with the knowledge and instructions needed to effectively collaborate with Erik on this project.

## When to Reference
- Starting a new AI session (read this first)
- Understanding how to work with Erik
- Finding role-specific instructions
- Looking for prompt templates and examples

## Contents
- **General prompt_library.md** - Standard prompts including the "Getting Started" prompt
- **ai_agents/** - Role-specific instructions and configurations for different AI agent types

## Usage Note
AI assistants should read the main README.md first, then check here for specific working instructions and role guidance.
